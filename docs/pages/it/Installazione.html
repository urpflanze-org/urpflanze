Installazione

Inclusione tramite CDN

Il modo più immediato per includere Urpflanze nel tuo progetto è utilizzare una versione hostata online. 
<!-- Link all'ultima versione completa minificata -->
<script src="https://cdn.jsdelivr.net/npm/@urpflanze/js"></script>
<!-- Link customizzabile -->
<script src="https://cdn.jsdelivr.net/npm/@urpflanze/js[@version]/build/umd/urpflanze[-light][.min].js"></script>

<!-- ES Modules -->
<script type="module">
	import * as Urpflanze from 'https://esm.run/@urpflanze/js'
</script>
Versione completa (UMD) Tutte le versioni

Inclusione tramite download

Se lo preferisci, puoi scaricare la versione completa, salvarla da qualche parte nel tuo progetto ed includerla nel tuo codice. 
<!-- Versione completa -->
<script src="path-to/urpflanze.js"></script>

<!-- Versione minificata-->
<script src="path-to/urpflanze.min.js"></script>
Se sei agli inizi e non hai bisogno di tutte le funzionalità puoi scaricare la versione light 
<!-- Versione light -->
<script src="path-to/urpflanze-light.js"></script>

<!-- Versione light minificata-->
<script src="path-to/urpflanze-light.min.js"></script>

Installazione tramite npm

Per installarla basterà lanciare il comando: 
npm i @urpflanze/js --save
Al termine potai includere Urpflanze nel tuo codice 
/**
 * Importazione completa
 */
import * as Urpflanze from '@urpflanze/js'

const scene = new Urpflanze.Scene()

/**
 * Importazione selettiva
 */
import { Scene } from '@urpflanze/js'

const scene = new Scene()
Per includere la versione light utilizza @urpflanze/js/dist/cjs/index-light
